
# 🥷 Urban Translate

Urban Translate is here to save you from getting lost in the ever-evolving language of the internet and real-world slang. Think of it as a tiny streetwise companion that turns slang and urban phrases into plain English, so you’re never out of the loop!


## Run Locally

Clone the project

```bash
  git clone https://github.com/1etu/urban-translate
```

Go to the project directory

```bash
  cd urban-translate
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

**or**

Clone the project

```bash
  git clone https://github.com/1etu/urban-translate
```

Go to the project directory

```bash
  cd urban-translate
```

Run the script

```bash
  npm run start:app
```
